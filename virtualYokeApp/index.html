Why do I get this warning?

TL;DR

This program uses your computer as a server to get your live data from your phone, and to get it in a secure way, a self signed certificate is used. Browsers are made to warn about this, but because the server is run on your computer, no risk is posed for you

Detailed Explanation

To get live data from your phone to your computer, this program opens an HTTPS server and communicates with the phone with websockets - a technology used to send data in real-time between a device and a server. In this case, your device is the client, while your computer opens a local server. HTTPS stands for Hypertext Transfer Protocol Secure, that is a protocol used to transfer data in text form between devices on the internet. To achieve the secure status (HTTPS vs HTTPS), a server needs a certificate, which validates the security of the server. Because this program opens a local server on your computer, a new self signed certificate is issued every time the app is opened, thus triggering a warning in your browser. Because the server is on your computer and does not communicate with any other server, your data is safe.
